Before importing the alt code into SAP Data Services repository, the following steps should be done carefully.
1. Read note 2239701, https://service.sap.com/sap/support/notes/2239701.
2. Read Quick Guide document attached in the note.
3. Pay attention to known issues and solution mentioned in the note.
4. If you encounter errors please check that you have downloaded the latest content attached to this note to ensure you are working with the most current version 
5. Open the OSS/CSN message with the detailed error description to the right component SV-RDS-EIM for the future issue
